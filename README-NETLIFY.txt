طريقة الرفع الصحيحة على Netlify
===============================

1) فك ضغط ملف muaini-netlify-fixed.zip.
2) ستجد index.html و service-worker.js في نفس المستوى.
3) ارفع محتويات المجلد كلها معًا إلى Netlify، ولا ترفع index.html منفردًا.
4) بعد النشر افتح:
   https://اسم-موقعك.netlify.app/service-worker.js
   يجب أن يظهر كود JavaScript وليس 404.
5) افتح الموقع واضغط Ctrl + F5 ثم فعّل الإشعارات.

ملاحظة:
- أضف logo.jpg و amr.jpeg إلى نفس المجلد إن أردت ظهور الشعار والصورة الشخصية.
- الإشعارات المحلية تعمل بعد تسجيل Service Worker.
- Push عند إغلاق الموقع تمامًا يحتاج خادم Push ومفاتيح VAPID، وليس موقع Netlify ثابتًا فقط.

الأصوات الجاهزة المضافة:
- أذان القاهرة — الشيخ محمد علي البنا
- أذان الفجر — الشيخ مشاري راشد العفاسي
- أذان تركيا — النسخة الثانية
- أذان جورجيا

مهم: ارفع مجلد audio مع باقي الملفات. عدم رفعه سيؤدي إلى عدم تشغيل الأصوات المحلية.
